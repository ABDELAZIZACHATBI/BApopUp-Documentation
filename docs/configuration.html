Configuration

Global Settings

Configure global settings that apply to all popups:

Cookie Settings

Control how often popups appear to the same visitor:

  • Every time - Show on every visit
  • 1 day - Once per day
  • 1 week - Once per week
  • 1 month - Once per month
  • 1 year - Once per year

Custom CSS & JS

Add custom styles and scripts globally:

/* Custom CSS Example */
            .bapopup-container {
                border-radius: 10px;
                box-shadow: 0 4px 6px rgba(0,0,0,0.1);
            }

Individual Popup Settings

Display Duration

Control when your popup is active:

  • Start Date - When to start showing the popup
  • End Date - When to stop showing the popup
  • Always Show - Ignore date restrictions

Trigger Settings

Choose how your popup appears:

  • Immediate - Show instantly
  • Delay - Wait specified seconds
  • Scroll - Show after scrolling
  • Exit Intent - Show when leaving page

Display Location

Control where your popup appears:

  • All Pages - Show everywhere
  • Homepage - Only on homepage
  • All Posts - On all posts
  • All Pages - On all pages
  • Specific Pages - Choose specific pages
