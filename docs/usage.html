Basic Usage

Creating Your First Popup

1. Create New Popup

Go to BApopUp > Add New in your WordPress admin.

2. Add Content

Use WordPress editor or Elementor to design your popup:

  • Add text, images, forms
  • Style your content
  • Add buttons and links

3. Configure Settings

Set up your popup behavior:

  • Choose trigger type
  • Set display duration
  • Select display locations

4. Publish

Click "Publish" to make your popup live.

Example Scenarios

Newsletter Signup

// Add form shortcode to popup
            [contact-form-7 id="123" title="Newsletter Signup"]

Special Offer

  • Set trigger to "Exit Intent"
  • Add coupon code
  • Set cookie duration to "1 week"

Welcome Message

  • Set trigger to "Immediate"
  • Display on homepage only
  • Set cookie duration to "1 month"
