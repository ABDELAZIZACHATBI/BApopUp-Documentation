Age Verification Popup

Adding Age Verification Button

Option 1: Using Elementor

If you're using Elementor page builder:

  1. Open popup in Elementor editor
  2. Find "Age Verification Button" in BA Popup widgets
  3. Drag and drop the button to your popup
  4. Customize appearance using Elementor controls:
    • Button text
    • Colors
    • Typography
    • Padding & margins

ðŸ’¡ The button from Elementor widget handles age verification automatically.

Option 2: Without Elementor

If you're using the default WordPress editor:


                <button onclick="BApopUp.confirmAge(this)">
                    I am over 18
                </button>
                
                
                <button onclick="BApopUp.confirmAge(this)" class="custom-style">
                    Confirm Age
                </button>
                
                <style>
                .custom-style {
                    background: #4CAF50;
                    color: white;
                    padding: 10px 20px;
                    border: none;
                    border-radius: 5px;
                    cursor: pointer;
                }
                .custom-style:hover {
                    background: #45a049;
                }
                </style>

ðŸ’¡ Add onclick="BApopUp.confirmAge(this)" to any button to handle age verification.

Key Details

  • Always use BApopUp.confirmAge(this) for manual buttons
  • No need for special classes - just the onclick handler
  • Cookie handling is managed by the confirmAge function
  • Works with any HTML element that can handle click events

Additional Examples


                <a href="#" onclick="BApopUp.confirmAge(this); return false;">
                    Verify Age
                </a>
                
                
                <div onclick="BApopUp.confirmAge(this)" class="custom-button">
                    Click to Verify Age
