Trigger Types

Available Triggers

Immediate

Shows popup immediately when page loads.

// Example settings
            Trigger Type: Immediate

Time Delay

Shows popup after specified number of seconds.

// Example settings
            Trigger Type: Delay
            Delay: 5 seconds

On Scroll

Shows popup when user scrolls to certain point of page.

// Example settings
            Trigger Type: Scroll
            // Appears at 50% scroll

Exit Intent

Shows popup when user moves mouse to leave page.

// Example settings
            Trigger Type: Exit Intent
